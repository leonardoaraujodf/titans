\begin{resumo}
Este documento apresenta como foi realizado o projeto de um robô autônomo seguidor de ectoplasma (substância fluídica que os fantasmas deixam aonde passam) cuja função é seguir, encontrar e capturar fantasmas. Este mostra de forma detalhada como foi realizado o projeto e o teste experimental. O microcontrolador utilizado foi o MSP430\cite{davies2008msp430} modelo g2553.
\end{resumo}
